# Daily — the operating log

`YYYY-MM-DD.md` per day: morning brief at top, session logs appended, open loops
at bottom. Weekly: `YYYY-Www-review.md`. Session logs include topics, decisions,
files changed, open loops — written by /daily save, scannable in 20 seconds.
One-line activity entries from client scope land under `## Activity`.
This folder is the firm-wide view; personal day logs live in
`Team/{person}/daily/` when someone wants their own lane.
