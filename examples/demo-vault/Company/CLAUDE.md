`reviewed:` updated whenever a human confirms contents.

| File | Holds |
|---|---|
| profile.md | identity, principles, how we operate, one-liner |
| offers.md | services, packages, pricing, delivery model |
| icp.md | who we serve: roles, pains, buying triggers, real examples |
| brand.md | voice rules, positioning, enemy, words to avoid |
| strategy.md | goals, current quarter, owners |
| team.md | (agency) roster summary — detail lives in Team/ |
| stack.md | tools where decisions/work actually live |
| market/ | competitor and market notes, one file per subject |
| standards/ | service quality bars — the ONLY Company files readable from client scope besides offers.md |

Client-specific facts never live here — they go in `Clients/{slug}/context/`.
