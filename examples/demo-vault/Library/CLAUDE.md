# Library — reusable assets

The firm's productizable kit. If it's useful twice, it lives here.

| Folder | Holds |
|---|---|
| playbooks/ | how we deliver each service, step by step, with quality bars |
| templates/ | document skeletons: reports, proposals, briefs, meeting notes |
| prompts/ | reusable prompts worth keeping |
| styles/ | output styles: how to write an email, report, proposal, post |
| swipe/ | external reference material worth imitating |

Hard rule: nothing in Library/ may link to or quote client folder content. Client
lessons enter only via /client harvest — anonymized. This keeps the entire
folder shippable as a product at any moment.
